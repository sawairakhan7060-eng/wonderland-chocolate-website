## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files (keep the `images/` folder intact).
3. Go to **Settings → Pages**.
4. Source: **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → Save.
5. Your site will be live at `https://<username>.github.io/<repo>/` in ~1 minute.

## Customise
- Update phone number `+923001234567` in `index.html` and `script.js`.
- Replace images in `/images` with your own (keep filenames).
- Edit prices/cakes directly in `index.html`.
